MeTime is a web application for online booking of services in a beauty salon. The user can choose a procedure, a master, a convenient date and an available time, then fill in their data and confirm the appointment. The application works as a Single Page Application with convenient navigation between pages.

The project was implemented using HTML, JavaScript, React and React Router to build the structure and routing, Redux to store booking data, Day.js to work with dates and times, as well as MUI (Material UI) to create a modern responsive interface.

The interface is fully adapted for mobile devices, tablets and desktops, which provides a convenient user experience on any screen.
